# Assignment 5: Low Fidelity Prototype

## Overview 

### Purpose
Low fidelity prototypes are created due to how quick and rapid representation of a user flow/product. It allows for UX designers to recieve feedback and allows them to make changes before time and money are put into the product. Lo-Fi prototypes are usually created through hand-drawn sketches and allows for the designer to maximize the amount of improvements before further work is done.
